📌 Spring Boot 学习笔记

什么是 Spring Boot？

Spring Boot 是一个用于构建独立、生产级 Spring 应用的框架。它简化了配置，让开发者能快速启动项目。

核心特性

  • 自动配置：根据项目依赖自动配置 Spring 应用
  • 起步依赖：简化 Maven/Gradle 配置
  • 嵌入式服务器：内置 Tomcat，无需部署 WAR 包
  • 生产就绪：提供健康检查、指标监控等功能
